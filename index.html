jmaps by digitalspaghetti

jMaps is a jQuery plugin that provides a really easy API to create and manage multiple google maps on any page.

As of January 2010 jMaps is currently in a lapsed state, however I am making some upcoming changes to ensure bugs are fixed and the current version works well with the current version of the of the Google Maps API. Added features will come later.
